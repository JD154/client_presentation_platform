# Client Presentation Platform

A modern web application built with React, TypeScript, and Vite for creating and displaying client presentations.

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS
- **UI Components**: Custom components with Radix UI
- **Build Tool**: Vite with Hot Module Replacement (HMR)

## Features

- Interactive client presentation slides
- Project management interface
- Responsive design
- Modern React patterns with TypeScript
- ESLint configuration for code quality

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `src/components/` - React components
  - `client/` - Client-specific components
  - `layout/` - Layout components
  - `presentations/` - Presentation slide components
  - `ui/` - Reusable UI components
- `src/lib/` - Utilities and data management
- `src/pages/` - Page components
- `data/` - Static data files
